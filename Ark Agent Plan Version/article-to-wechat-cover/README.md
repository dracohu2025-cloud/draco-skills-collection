# 飞书文档 → 公众号封面图（Ark Plan 专用版）

基于 Ark Agent Plan 原生 Seedream 生图能力，自动为飞书文档生成匹配的公众号封面图。

## 功能特性

- ✅ 自动读取飞书文档内容，提取文章主题
- ✅ 调用 Seedream 生成 2.35:1 比例的公众号封面
- ✅ 支持自动插入到飞书文档顶部作为 Hero 图
- ✅ 支持上传到公众号素材库
- ✅ 内置封面图设计最佳实践（尺寸、构图、字体）

## 与通用版本的区别

| 通用版本 | Ark Plan 专用版本 |
|---|---|
| 使用 Nano Banana / Gemini 生图 | 使用原生 Seedream 生图 |
| 依赖 OpenRouter API Key | 无需额外配置，使用 Agent Plan 原生能力 |

## 使用场景

1. 写完飞书文档后，一键生成公众号封面
2. 批量为多篇文章生成统一风格的封面
3. 快速迭代封面设计，对比不同方案

## 核心流程

```
飞书文档 URL
    ↓
读取文档内容
    ↓
提取文章主题、关键词、风格
    ↓
生成 Seedream prompt
    ↓
调用 Seedream 生成 2.35:1 封面
    ↓
[可选] 插入飞书文档顶部
[可选] 上传公众号素材库
    ↓
返回图片结果
```

## Prompt 模板

```
设计一张微信公众号封面图，比例 2.35:1

文章主题：{文章主题}
核心关键词：{关键词1, 关键词2, 关键词3}
文章风格：{科技/教育/商业/人文/生活}

要求：
- 现代简约风格，视觉冲击力强
- 标题区域留白（顶部 20%）
- 主题明确，一眼就能看懂文章在讲什么
- 色彩搭配专业，符合公众号审美
- 避免过多文字，以图形符号为主
```
