# 飞书文档 → 公众号封面图（Ark Plan 专用）

## 触发条件

当用户说：
- "给这篇文档生成封面"
- "生成公众号封面"
- "文章封面图"
- "帮我做个封面"

## 前置检查

- ✅ 用户提供了飞书文档链接
- ✅ 确认是微信公众号封面（2.35:1 比例）
- ✅ Seedream 服务可用

## 执行步骤

### 1. 读取飞书文档

调用 `feishu.fetch_doc` 工具读取文档内容：
- 标题
- 摘要/前言
- 核心关键词
- 文章风格（技术/商业/生活/教育等）

### 2. 分析并生成 Prompt

基于文档内容，生成 Seedream prompt：

```
# 角色
你是专业的公众号封面设计师，精通 2.35:1 比例的封面设计

# 输入信息
文章标题：{title}
文章主题：{topic}
核心关键词：{keywords}
文章风格：{style}

# 输出要求
1. 生成一张 2.35:1 比例的公众号封面图
2. 风格现代简约，有视觉冲击力
3. 顶部 20% 区域留白（用于放置文章标题）
4. 用图形符号表达主题，避免过多文字
5. 色彩搭配专业，符合公众号审美

# 设计规范
- 尺寸：1280x544 像素（2.35:1）
- 主色调：与文章主题匹配
- 视觉焦点：画面中心偏下位置
- 标题区域：顶部 25% 区域保持简洁
```

### 3. 调用 Seedream 生图

使用 Ark Plan 原生能力：
- 工具：`byted-seedream-image-generate`
- 参数：
  - `prompt`: 上面生成的设计 prompt
  - `width`: 1280
  - `height`: 544
  - `style`: 根据文章类型选择

### 4. 后处理（可选）

根据用户需求：
- **插入飞书文档**：调用飞书 API 在文档顶部插入图片
- **上传公众号**：调用微信公众平台 API 上传到素材库
- **下载本地**：保存到指定路径

## 输出模板

```
✅ 公众号封面图已生成

📄 文章：{文章标题}
🎨 设计关键词：{关键词1, 关键词2, 关键词3}
📐 尺寸：1280×544 (2.35:1)

[图片预览]

下一步操作：
- [ ] 插入到飞书文档顶部
- [ ] 上传到公众号素材库
- [ ] 重新生成（调整风格）
- [ ] 下载到本地
```

## 常见优化方向

| 文章类型 | 推荐风格 | 配色方案 |
|---|---|---|
| 技术/AI | 科技感、未来风 | 蓝、紫、渐变 |
| 商业/职场 | 专业、简约 | 深蓝、灰、金 |
| 教育/知识 | 清新、明亮 | 绿、橙、白 |
| 生活/人文 | 温暖、有质感 | 暖色调、自然色 |
| 新闻/资讯 | 冲击力、简洁 | 黑、白、红 |
