# Handoff Map

选型完成后，直接进入对应 skill：

## Manim
进入：`manim-video`

下一步通常是：
- 写 narrative / 分镜
- 拆场景
- 先做最小 specimen scene
- 再补全视频

## Motion Canvas
进入：`motion-canvas`

下一步通常是：
- 定 scene 列表
- 写 `project.ts`
- 先做 1 个主 scene
- 在预览里调节奏

## Remotion
进入：`remotion`

下一步通常是：
- 定 composition
- 先做静音版
- 抽 still 验图
- 再补旁白和字幕

## 原则

选型回答结束后，不要停在抽象讨论里。立刻切到对应 skill 开工。
